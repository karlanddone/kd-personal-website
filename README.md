# Karl Schroth | K&D Personal Website

Source code for the personal portfolio of **Karl Schroth** (*Operational Systems Architect*). 

This repository contains a highly polished, single-page application built to showcase past projects, operational philosophies, and robust scalable systems.

## 🎨 Visual Identity: "Frontier Tech / Industrial Lo-Fi"
The aesthetic is designed to appeal to brilliant, highly logical engineers and aerospace/gaming founders. It eschews generic bubbly SaaS templates in favor of a sleek, dark-mode, terminal-inspired architecture. 
- **Palette:** Deep charcoal backgrounds with high-contrast Neon Cyan, Pink, and Industrial Orange accents.
- **Typography:** Space Grotesk (Headings) and Roboto Mono (Technical Copy).
- **UI Elements:** Subtle blueprint grids, exposed "schematic" borders, and cut-corner aesthetics designed with Tailwind CSS v4.

## 🚀 Tech Stack
- **Framework:** Vanilla JavaScript + HTML5
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)

## 🛠️ Local Development
To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/karlanddone/kd-personal-website.git
   ```
2. Navigate to the directory:
   ```bash
   cd "K&D Personal Website"
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
